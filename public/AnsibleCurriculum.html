Ansible Curriculum

Course objective :

After completion of this course, System admins will be able to deploy basic system configuration using Ansible in a agentless and server setup.

Prerequisite :

Familiarity with Linux/Nix system or and RCHSA certified skilled administrator.

Total Hours :

32

Course Modules :

 1.The Essentials of Configuration and Deployment
o What we can do with Ansible
o Our goals before, and our goals now

 2.Setting Up
o Configuration
o Inventory

o Playbooks
 Roles, Plays, And Tasks
 Variables and Modules

 3.Quick Examples
o Deploying configuration of Apache server
o Managing changes in the configuration

 4.Playbooks
o Roles
o Inclusion
o Variables
o Filters
o Conditionals
o Loops
o Accelerated Mode
o Asynchronous Actions
o Pooling
o Dry Run
o Delegation, Rolling Updates and Local Actions
o Error Handling
o Lookups
